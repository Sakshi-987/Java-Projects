## Current Project

### 🎮 Mad Libs Game

A simple interactive console-based game that generates a short and humorous story based on user-provided words. The application prompts the user to enter different types of words such as adjectives, nouns, and verbs, then dynamically inserts them into a predefined story template.

This project was built to practice user input handling, string manipulation, variable usage, and basic program flow in Java while creating an engaging beginner-friendly application.

## Getting Started

### Prerequisites

* Java JDK 8 or higher
* Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or terminal

### Run the Project

Compile the program:

```bash
javac MADLIBSgame.java
```

Run the program:

```bash
java MADLIBSgame
```
